FindInotify¶

Try to find inotify on this system. This finds:
  • libinotify on Unix like systems, or
  • the kernel’s inotify on Linux systems.

This will define the following variables:

Inotify_FOUND
True if inotify is available
Inotify_LIBRARIES
This has to be passed to target_link_libraries()
Inotify_INCLUDE_DIRS
This has to be passed to target_include_directories()

On Linux, the libraries and include directories are empty, even though Inotify_FOUND may be set to TRUE. This is because no special includes or libraries are needed. On other systems these may be needed to use inotify.

Since 5.32.0.
