ECMQMLModulesÂ¶

Find QML import modules through a find_qmlmodule() call. It uses the qmlplugindump application to find the plugins and sets them up as runtime dependencies.

This is useful so that when we configure a project we are noified when some QML imports are not present in the system, thus having the application compilable but fail at runtime.

ecm_find_qmlmodule(<module_name> <version>...)

Any further arguments passed will be forwarded into a find_package() call. See find_package() documentation for more information.

Usage example:

ecm_find_qmlmodule(org.kde.kirigami 2.1)

Since 5.38.0.
